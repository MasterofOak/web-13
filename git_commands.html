Create a Repository

From scratch -- Create a new local repository:
$ git init [project name]
Download from an existing repository:
$ git clone my_url

Observe your Repository

List new or modified files not yet commited
$ git status
Show the changes to files not yet staged
$ git diff
Show the changes to staged files
$ git diff --cache
Show all staged and unstagged file changes
$ git diff HEAD
Show cahnges between 2 commits ids
$ git diff commit1 commit2
List the change dates and authors for a file
$ git blame [file]
List the change dates and authors for a file
$ git show [commit] : [file]
Show full change history
$ git log
Show changes history for file/dir including diffs
$ git log -p [file/dir]

Working with branches

List all branches
$ git branch
List all branches local and remote
$ git branch -av
Switch branch
$ git checkout my_branch
Create a new branch
$ git branch branch_name
Delete a branch
$ git branch -d branch_name
Merge branch_a into branch_b
$ git checkout branch_b
$ git merge branch_a
Tag the current commit
$ git tag my_tag

Make a change

Stage a file for commit
$ git add [file]
Stage all files in current dir
$ git add .
Tag the current commit
$ git commit -m "message"
Commit all tracked files
$ git commit -am "message"
Unstage a file
$ git reset
Revert everything to the last commit
$ git reset --hard

Syncronize

Get latest changes from origin
$ git fetch
Get latest changes from origin and merge
$ git pull
Fetch latest changes from origin and rebase
$ git pull --rebase
Push changes to origin
$ git push

Finally

When in doubt use git help
$ git command --help
